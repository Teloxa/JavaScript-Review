# JavaScript Review — Intro Exercises

This repository contains short JavaScript examples and solved practice problems to help you review core language features and common algorithmic tasks.

## Files
- [00_Introduccion.js](00_Introduccion.js): simple examples and small exercises covering:
	- Variables and data types
	- Function declaration and arrow functions
	- Arrays and common methods (`map`, `filter`, `reduce`)
	- Objects and destructuring
	- Asynchronous patterns (Promises, `async/await`)
	- Error handling with `try/catch`
	- A commented DOM example for browser use
	- A FizzBuzz small exercise
	- Notes about modules and coding tips
- [01_Problems.js](01_Problems.js): solved algorithmic problems with demo outputs:
	- `twoSum` — indices for two-sum problem
	- `reverseString` — reverse a string
	- `isPalindrome` — palindrome check (ignores non-alphanumeric)
	- `factorial` — iterative factorial
	- `mergeSortedArrays` — merge two sorted arrays
	- `flattenArray` — flatten nested arrays

## How to run
Run the example/demo files with Node.js from the repository root:

```bash
node 00_Introduccion.js
node 01_Problems.js
```

Output will be printed to the console for each demo.
